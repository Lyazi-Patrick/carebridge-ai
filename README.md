# CareBridge AI

CareBridge AI is a verified medical fundraising and patient-support platform built for the OpenAI Build Week Hackathon.

## Project structure

- `frontend/` — React 19, Vite, Tailwind CSS
- `backend/` — Express, Prisma, PostgreSQL integration
- `shared/` — shared Zod contracts and TypeScript types
- `docs/` — project documentation

See [local development instructions](docs/local-development.md) to run the connected application.
